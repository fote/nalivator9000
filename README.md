# How to build

1. Install Golang
2. Install govendor:
```go get -u github.com/kardianos/govendor```
3. Get the source:
```go get github.com/fote/nalivator9000```
4. Now compile:
```cd $GOPATH/src/github.com/fote/nalivator9000 && go build -v```
